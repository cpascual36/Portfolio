Projects

  • ACSSA

    ACSSA wanted to reach international property developers who are interested in investment property in the Dominican Republic. I worked with ACSSA to create a responsive website providing an optimal interactive experience across a range of devices while showcasing the stunning property. The website incorporates the Google Maps API allowing visitors to pinpoint the property's location and view the property in map or satellite view. A gallery of images highlights the property's spectacular location.

    The website utilizes the bootstrap framework, JQuery, and JavaScript. Photoshop was used to crop, color correct and optimize the images for the web.

  • Neighborhood Map

    This single-page application featuring a map Tampa, FL. Map markers identify popular locations and a listview allows you to browse all locations or filter by name. Clicking on a marker provides the user with additional information about each location. The application is responsive and can easily by used on desktop, tablet or phone. The application makes use of third-party APIs (specifically GoogleMaps API and Wikipedia API) and uses JavaScript, Jquery, Bootstrap framework, Knockout (MVVM) framework and AJAX requests to retrieve information from the Wikipedia API.

  • Website Optimization

    The goal of this project was to optimize the critical rendering path of an existing website to achieve a PageSpeed Insights score above 90. To achieve this goal inefficiencies in the website's scroll animation were eliminated. The site now scrolls at 60 frames per second. This project was completed as part of the Udacity Front End Developer Nanodegree program. The technologies used include Google Chrome DevTools and task automation with Gulp.
